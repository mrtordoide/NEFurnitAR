# NEFurnitAR

    A full fledge furniture business application with products which can be viewed in AR in 
    supported devices.

#### Components:

    - Android Application
    - Firebase Cloud Store Database
    - Google ARCore and Sceneform SDK
    - 3D models from Google Poly
